如何制作 Win To Go(WTG)

2025-05-11

💿你的U盘/SSD上将会有一台电脑！
💼随身携带，外出无忧

Win To go 介绍

win to go 的全称为Windows to go，是一种免安装的Windows 系统（安装在U盘或SSD固态硬盘上）

它可以在任何电脑上运行，并且可以运行任何 Windows 应用程序，包括Windows 10、Windows 8、Windows 7等。

准备工作

  1. 准备一个U盘或SSD固态硬盘，容量至少为8GB
  2. 准备一个Windows系统镜像文件，例如Windows 10镜像文件
  3. 准备一个Win To Go制作工具/写盘工具，例如 Rufus

制作Win To Go

  1. 打开Win To Go制作工具/写盘工具，选择 U盘 或 SSD固态硬盘

  2. 选择一个 Windows 系统镜像文件（Win 8 以上）

  3. 选择引导模式，一般选择UEFI
    a）如何查看电脑的引导模式 ？

    win + r 打开 运行 ，输入 msinfo32 ，回车即可打开 系统信息 窗口，在 系统信息 窗口中，找到 系统类型/Bios ，即可查看电脑的引导模式。

  4. 打开所需软件（Rufus）
    选择好SSD固态硬盘后，点击 选择 按钮，选择 Windows系统镜像文件

    然后点击 启动 按钮，选择 分区类型 ，其他的保持默认即可

    点击开始制作即可开始制作Win To Go 等待制作完成即可

  5. 制作完成后，将U盘或SSD固态硬盘插入电脑，进入BIOS
    【BIOS主板进入方法/快捷键】

  • 进入BIOS后，找到启动选项，将U盘或SSD固态硬盘设置为第一启动项
