Github是什么？

Github是一个全球性开源平台。在此平台上有适合不同人群的不同大量优质开源资源及开源作品——Github一直与微软（Microsoft）有着亲密的关系，是微软的长期合作伙伴，安全性双重保障，可以放心使用！Github主要面向开发者群体，所谓开源就是开放源码的意思。
